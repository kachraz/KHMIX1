```sh 
df -h 
```

```sh 
❯ df -h

tmpfs            64M     0   64M   0% /dev
shm              64M     0   64M   0% /dev/shm

/dev/sdb1        44G  304K   42G   1% /tmp
/dev/loop3       32G   18G   12G  61% /workspaces
```
